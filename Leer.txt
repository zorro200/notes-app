El propósito de realizar este proyecto es el de adquirir conocimientos trabajando en React y muchas más
tecnologías nuevas, muy buenas e interesantes. Creando una buena app full-stack.

Para saber las tencologías utilizadas, tanto de desarrollo como no, se puede ver el archivo "package.json"

Este proyecto comenzó a raíz de Redux de Midudev y fue creciendo en función avanzaba el Bootcampo fullstack.

Este proyecto prentende tener prácticamente todos los conocimientos que contiene el Bootcamp, por lo que se irá
actualizando el contenido a medida que se avance en el Bootcamp o se retroceda en este para adquirir conocimientos
o funcionalidades nuevas y que deberían estar.
